CIS 2023 Linux Final Project

Creating a bash script that prints numbers 1-50 in the terminal when exicuted.

Purpose: Creating project: Used mkdir to create a directory for the project. Used git init to
create an empty repository in the directory.



Creating Script: used nano to create file, used chmod to make file executable.

Linking to Github: Used ssh-keygen to create a shared key. Added it to Github account.

Staging and Commit: Used git add to move files to staging area, and used git commit to save a
snapshot of the staged changes.

Pushing: Used git push to upload the staged files to public Github repo.
